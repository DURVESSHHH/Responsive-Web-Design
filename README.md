# Responsive Web Design Prototype

A single-file, zero-dependency responsive prototype demonstrating modern CSS layout techniques — fluid grids, flexible images, and `clamp()`-based typography — paired with JavaScript-driven responsive navigation.

## Features
- **Fluid grid** using CSS Grid `auto-fill` + `minmax()` for organic reflow
- **Clamp-based typography** that scales continuously from mobile to 4K
- **4 breakpoints**: wide desktop (≥1400px), desktop, tablet (≤900px), mobile (≤640px), narrow mobile (≤400px)
- **Responsive nav** with a sticky blur header, animated hamburger, and full-height mobile drawer
- **Accessibility**: keyboard-navigable, `aria` attributes, `prefers-reduced-motion` support
- **No frameworks** — semantic HTML5, vanilla CSS custom properties, and ~50 lines of JS
